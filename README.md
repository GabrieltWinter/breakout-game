Breakout Game
Jogo de Breakout desenvolvido com HTML5 Canvas e JavaScript puro. Conta com sistema de power-ups, partículas, combo de pontos e múltiplos níveis.
🔗 Demo
Ver online
🛠 Tecnologias

HTML5 Canvas
CSS3
JavaScript

⚙️ Funcionalidades

Controle por teclado (← → / A D), mouse e clique
4 power-ups: paddle largo, bola lenta, fireball e +1 vida
Sistema de combo com multiplicador de pontos
Partículas ao quebrar tijolos
Tijolos com 2 níveis de vida nas linhas superiores
Dificuldade aumenta a cada nível

🎮 Como jogar

Espaço ou clique para iniciar
← → ou A D para mover o paddle
Mouse para controle por movimento
Colete os power-ups que caem ao quebrar tijolos marcados com ★

📦 Como rodar localmente
bashgit clone https://github.com/GabrieltWinter/breakout-game.git
Abra o index.html no navegador.
